# Stack du projet

# Backend
Langage : Java 21
Framework : Spring Boot (gestion via Maven)
Gestion des dépendances et compilation : Maven 21

# Frontend
Technologie : JSP (Java Server Pages)
Template / pages web servies par Spring Boot

# Base de données
Système : PostgreSQL 17
Connexion via Spring Data JPA / Hibernate

Serveur web intégré

Tomcat intégré à Spring Boot (par défaut, http://localhost:8082)
